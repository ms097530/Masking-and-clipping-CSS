How to clip and mask images

Linear gradient mask

#img-area-1 { -webkit-mask-image: linear-gradient(black, transparent); mask-image: linear-gradient(black, transparent); }

Mask with PNG with transparent background

#img-area-2 { -webkit-mask-image: url('./public//m-transparent.png'); mask-image: url('./public/m-transparent.png'); }

Mask - paint splatter PNG

#img-area-3 { -webkit-mask-image: url('./public/splatter-1.png'); mask-image: url('./public/splatter-1.png'); -webkit-mask-size: cover; mask-size: cover; }

Mask - paint splatter SVG

#img-area-4 { -webkit-mask-image: url('./public/splatter-2.svg'); mask-image: url('./public/splatter-2.svg'); -webkit-mask-size: cover; mask-size: cover; -webkit-mask-position: center left; mask-position: center left; -webkit-mask-repeat: no-repeat; mask-repeat: no-repeat; }

Mask - butterfly PNG
